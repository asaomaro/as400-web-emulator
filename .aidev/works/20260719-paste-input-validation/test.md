## 未検証の穴

- **`StatusBar` のメッセージ優先（notice がホストの systemMessage を隠す）に
  自動テストが無い。** 変更は template の `v-if` 1 行で、既存の StatusBar テストが
  無いため新規に足す必要がある。**実機確認に頼っている状態**であり、
  deliver の PR 本文へ引き継ぐ。
- **DBCS 欄の単一行ペースト**は従来経路のため、矩形折返し・横割り込みが効かない
  （decisions.md D1）。これは仕様上の制限であり不具合ではないが、
  自動テストで「効かないこと」を固定してはいない。

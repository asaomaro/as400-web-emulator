# 要件: プリンターセッションの DBCS（日本語）対応

## 背景 / 課題
SBCS プリンターセッション（PR #15）は完成したが、本 PJ は DBCS 中心であり、日本語帳票の受信が未対応。
親作業の design.md D6 で「DBCS は SBCS 完成後の増分。参照実装なし＋実 SCS 未採取」と保留していた。

## 目的 / ゴール
DBCS（CCSID 1399・日本語）のスプールを受信し、SCS 中の SO/SI 付き全角を桁揃えを保って帳票化する。

## スコープ
### 対象
- ScsDecoder の DBCS 対応（SO/SI シフト、`decodeDbcsPair`、全角=2 桁）。
- DBCS プリンター接続（端末タイプ・CCSID 申告）の実機確認。
- 実機 DBCS スプールでの end-to-end 検証。
### 対象外
- 自動 PDF/自動印刷（親作業と同じく後続）。

## 完了条件（受け入れ基準）
- [ ] CCSID 1399 でプリンター接続でき、DBCS スプールを受信できる。
- [ ] SCS 中の SO/SI 付き全角が正しくデコードされ、桁が揃う。
- [ ] 実採取 DBCS スプールの golden テストと実機検証が green。

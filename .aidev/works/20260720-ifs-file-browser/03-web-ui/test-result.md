## ラウンド 2: review 後の再検証

review で **must 8 / should 9 / nit 13** を受けて修正した。詳細は `review.md`、判断は `decisions.md`。

**最も重い指摘は「ツリーを作っていないのに完了と記録していた」**（M7）。
実装したうえで、E2E にツリー操作の 2 項目を足した。

  **アップロードが動かなくなった**。jsdom では再現しない挙動

追加したテスト（web-ui 491 → 497）:

- フォルダをクリックすると開く（プレビューしにいかない）
